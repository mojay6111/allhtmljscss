Section 9: Double Click Event Example

The ondblclick event triggers when an element is double-clicked. This can be useful for things like highlighting, zooming, or opening extra details.

Section 10: Mouse Over & Mouse Out Events Example

The mouseover event triggers when the mouse pointer enters an element, and the mouseout event triggers when the pointer leaves. These are useful for interactive UI effects such as highlighting menus, tooltips, or changing button styles when a user hovers over them.
